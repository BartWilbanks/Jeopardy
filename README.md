# Jeopardy Live (3000-question local bank)

## Run locally
1. Install Node.js (18+)
2. In this folder:
   - `npm install`
   - `npm start`
3. Open:
   - Host: http://localhost:3000/host.html
   - Player: http://localhost:3000/player.html

## Deploy to Render
- Build command: `npm install`
- Start command: `npm start`

## Question bank
- `public/questions_3000.json` (3000 items)
- Debug counts: `/debug/bank`
